Code For Marriageの評判・料金｜エンジニア専門結婚相談所の実態

最終更新: 2026年7月（料金は2026年7月に公式サイトで確認）

結論：Code For Marriageはこんな人に向く

  • ●特徴：ITエンジニア専門×仲人が全員元SE×100%リモートという、他にない組み合わせの相談所。IBJ正規加盟で出会いの母数は全国規模です。
  • ●料金：入会77,000円＋月11,000円＋成婚費220,000円（税込）。1年で成婚した場合の総額目安は約42.9万円。U-30プランなら若手はさらに割安です。
  • ●向く人：仕事や技術の話を「説明しなくても分かってくれる」担当者が欲しいエンジニア。地方・リモート勤務で通所が難しい人。
  • ●注意点：2022年設立で第三者口コミが少ないため、無料相談での相性確認が判断の中心になります。

※本ページの料金等は公式サイトの公表情報（2026年7月確認・税込）にもとづきます。最新かつ正確な数値・契約条件は公式サイトおよび無料相談で必ずご確認ください。

Code For Marriageの強み・特徴

日本初を掲げるITエンジニア専門の結婚相談所

2022年設立で、対象をITエンジニア・IT職に特化。エンジニアの働き方（リモート・裁量労働・技術の話題）を前提にした婚活支援を掲げており、「趣味や仕事を理解してもらえない」という婚活のミスマッチを減らしやすい設計です。

仲人（カウンセラー）が全員元SE

公式サイトによれば、担当する仲人は全員がシステムエンジニア出身。職務内容や繁忙期の事情を説明なしで理解してもらえるのは、他の相談所にはない強みです。技術職特有の「自己PRの言語化」も相談しやすい環境といえます。

100%リモート対応で全国から利用できる

カウンセリングから活動サポートまでオンラインで完結します。所在地は東京ですが、地方在住のエンジニアやフルリモート勤務者でも利用可能です（お見合い自体はIBJの仕組みで対面・オンライン両方があります）。

IBJ正規加盟＋U-30向けの割安プラン

IBJ（日本結婚相談所連盟）の正規加盟店で、お相手探しは全国規模の会員ネットワークが対象。30歳未満向けのU-30プラン（入会33,000円・月7,700円）があり、若手エンジニアは初期負担を抑えて始められます。

  • +U-30プランで若手は初期費用・月会費を抑えられる
  • +料金体系がシンプルで公式サイトに明記されている

デメリット・注意点

  • -2022年設立と新しく、第三者の口コミがWeb上にほとんどない
  • -成婚料220,000円が成婚退会時に発生する
  • -エンジニア専門のため、他業種の相談ノウハウを求める人には不向き
  • -対面でじっくり会って相談したい人にはリモート中心が合わない場合がある

口コミがまだ少ない専門相談所の見極め方（正直な解説）

当サイトの調査（2026年7月時点）では、Code For Marriageの第三者による口コミ・体験談はWeb上にまだほとんど存在しません。当サイトは実在しない口コミを作成・掲載しない方針のため、代わりにエンジニアが専門特化型の相談所を見極めるポイントを整理します。

  • 1.「元SE」の具体度を無料相談で確認：担当仲人の経歴（開発経験・業界）を聞き、自分の職種・働き方をどこまで理解して提案してくれるかを見ます。
  • 2.IBJ加盟という客観的事実を土台にする：出会いの母数はIBJネットワークに依存するため、相談所の規模より「サポートの質」で選ぶのが専門特化型の正しい評価軸です。
  • 3.リモート完結の相性を試す：無料相談自体がオンラインなので、「画面越しで本音を話せるか」をそのままテストできます。
  • 4.契約条件を書面で確認：中途退会時の精算・休会制度・成婚の定義（IBJ準拠か）は必ず書面で確認しましょう。

よくある質問

Code For Marriageの料金は総額いくらかかりますか？+
公式公表の料金（税込・2026年7月確認）は、スタンダードプランで入会費77,000円＋月会費11,000円＋成婚費220,000円です。1年活動して成婚した場合の総額目安は約429,000円（77,000＋11,000×12＋220,000）。30歳未満はU-30プラン（入会33,000円・月7,700円）でさらに抑えられます。最新の料金は公式サイトでご確認ください。
エンジニア以外でも入会できますか？+
サービスはITエンジニア・IT職向けに設計されています。エンジニア以外の方の入会可否は公式サイトに明記がないため、無料相談でご確認ください。「エンジニアと結婚したい」お相手探しはIBJネットワーク全体が対象になります。
地方在住でも利用できますか？+
利用できます。公式サイトは100%リモート対応を明記しており、カウンセリングやサポートはオンラインで完結します。お見合いはIBJの仕組み上、対面・オンラインの両方があります。
口コミがほとんど見つからないのですが大丈夫ですか？+
2022年設立の小規模専門相談所のため、第三者の口コミはWeb上にまだほとんどありません（2026年7月時点の当サイト調査）。判断材料は、IBJ正規加盟という客観的事実、料金の透明性、無料相談での相性確認に置くのが現実的です。当サイトは実在しない口コミを掲載しない方針のため、体験談の紹介はしていません。
IBJ加盟とはどういう意味ですか？+
IBJ（日本結婚相談所連盟）は国内最大級の結婚相談所ネットワークで、加盟相談所の会員同士がお見合いできます。Code For Marriageは会社概要等でIBJ正規加盟を明記しており、小規模でも出会いの母数は全国規模です。
他のエンジニア向け婚活サービスとの違いは？+
マッチングアプリと違い、独身証明書等の提出を前提とする結婚相談所の仕組み（IBJ）の上で、仲人が伴走する点が特徴です。その中でも「仲人が全員元SE」という専門特化は珍しく、職業理解のあるサポートを重視する人に向いています。
